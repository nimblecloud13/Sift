Getting Started with Sift
Sift through your media quickly. No fuss.
1
Open a folder
Click Browse in the top bar or on the home screen, or paste a folder path directly into the path box and press Enter. Sift will scan and load all media files it finds. To include files in subfolders, use Include Subfolders instead. Large collections are loaded in chunks of 2,000 to keep things running smoothly. A button will appear in the filmstrip to load the next chunk when you're ready.
2
Set up your sift destinations
In the Sift Destinations panel on the right, you'll see two default folders. Save and Delete. Rename them or add up to 5 destinations using + Add. Each destination gets a number key (1–5).
3
Sift your files
Each file is displayed one at a time. Press the number key for the destination you want (1, 2, etc.) or click the button in the Sift panel. The file moves to that subfolder and Sift automatically advances to the next one.
4
Navigate freely
Use the ← → arrow keys, the on-screen arrows, or click any thumbnail in the filmstrip at the bottom to jump to any file. Already-sorted files show a colored badge. Click it to unsort and move them back.
5
Undo a move
Made a mistake? Click ↩ Undo Last Move in the Sift panel, or click the badge overlay on any sifted image to unsift it individually. Files are moved back to the original folder instantly.
6
Filter by media type
Use the Show buttons to limit the view to Images, Videos, Audio, or All. You can also change the list order between A→Z and Z→A from the List Order panel.
7
Keyboard shortcuts
1–5 sift to destination  ·  ←→ navigate  ·  Z undo  ·  B browse  ·  Space play/pause  ·  F fullscreen
Additional Features
●
Strip Metadata
Click ✂ Strip Metadata in the top bar to duplicate any folder with all embedded metadata scrubbed from every file. Including GPS location tags, camera make/model, timestamps, and other personally identifying information that gets baked into photos and videos without you knowing. The original folder is untouched; a new folder is created next to it with _no_metadata appended to the name. Images (JPG, PNG, WebP) are cleaned using the built-in image library. Video and audio files require ffmpeg. If it isn't installed, Sift will prompt you to install it automatically via Windows Package Manager (one click, takes about a minute), or you can skip those files for now and come back later.
●
Grid view
Click ⊞ Grid in the Grid Display panel to see all your loaded images at once in a full-window grid. Adjust the number of columns with the arrows or number field. Click any image to jump straight to it. Use the autoscroll button to browse hands-free, with an adjustable speed slider.
●
Slideshow mode
Just want to enjoy your collection without sifting? Hit ▶ Play in the Slideshow panel and Sift will automatically advance through your files. Adjust the interval with the seconds slider. Pair it with ⛶ Full to go fullscreen for a proper gallery experience.
●
Fullscreen
Press F or click ⛶ Full to expand the current file to fill your entire screen. The sidebar, filmstrip, and top bar all hide so nothing gets in the way. Press Escape or F again to return.
●
Video & audio playback speed
When viewing a video or audio file, use the Playback Speed buttons (1×, 2×, 3×, 4×) to speed through footage quickly. Great for scanning long clips. Click ★ Set as Default to remember your preferred speed across sessions.
●
Display on Launch
Choose whether Sift reopens your last folder automatically when it starts, or opens to a blank state. Set your preference using the Display on Launch toggle in the controls panel.
●
Light & dark mode
Switch between light and dark themes using the Theme toggle in the controls panel. Your preference is saved and applied automatically on next launch.
●
Copy Path
The Copy Path button in the status bar (centered below the selected thumbnail) copies the full file path of the current file to your clipboard. Handy for pasting into other apps.
